# Taha's Portfolio

hi my names taha and this is my portfolio

Live URL: [tahagillani.ca](https://tahagillani.ca)

## Tech Stack
* React (TypeScript)
* Tailwind CSS
* Framer Motion
* Matter.js
* Vite

## Instructions

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally
```bash
npm run dev
```

### 3. Build for production
```bash
npm run build
```
